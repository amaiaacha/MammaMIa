# MammaMia
 
## Descripción 
El proyecto Mamma Mia consta de varias partes. En un principio, en la página principal "home.html" vamos a encontrar dos maneras de buscar pizzas, la primera según el estilo de las masas: 
	 - Contamos con tres tipos de masas y si accedemos a cualquiera de ellas van a aparecer las pizzas asociadas a dichas masas y la información de cada masa.
 	 - Seguido, encontraremos una lista de ingredientes, en la cual nos indica cierta información sobre los mismos y también nos da la posibilidad de acceder a otra página 	 donde nos aparecen las pizzas que contiene el ingrediente que hemos seleccionado.
Por otro lado, en la barra de arriba, tenemos acceso directo a otros apartados, así como una vista general de todas las pizzas con los detalles que la componen (el tipo de masa y una lista de ingredientes), también un apartado donde hay un formulario para personalizar una pizza nueva al gusto del usuario y por último, un apartado que indica en que zona y que número de contacto tiene cada tienda física.
